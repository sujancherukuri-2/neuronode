# Video Analysis Script (4-7 min)

## 0:00 - 0:30 Introduction

- What the product is: an AI Second Brain that keeps notes alive and useful.
- The core value: answering questions with sources, not just summaries.

## 0:30 - 2:00 Product Walkthrough

- Dashboard: system overview, confidence, and activity trends.
- Notes: structured capture, tags, links, insights.

- UI built with Next.js and Tailwind.
- API routes for CRUD and query.
